# HomeHeros Admin Portal / CRM — QRD

## Scope & Platforms
- Web (Next.js/Vite + Vercel).  
- For Admin, CS, Finance, Contractor Managers.

## Roles
- Admin: full control.  
- Ops/CS: booking edits, reassignments, disputes.  
- Finance: payouts, reports.  
- Contractor Manager: manage Heros, documents, bookings.

## MVP Features
- Live Ops Board (bookings by state, Hero map, SLA alerts).  
- Booking Detail (timeline, GPS, media, payments, reassign).  
- User 360 (customer + Hero profiles, docs, flags).  
- Contractors management (SLAs, categories, coverage).  
- Payments & refunds (Stripe links).  
- Disputes workflow.  
- Reports (revenue, cancellations, acceptance rate).  
- CRM Lite (search, notes, tickets).

---

**Shared System Interactions**
- Central control point for Customers & Providers.  
- Can reassign jobs, issue refunds, manage disputes.  
- Contractor managers onboard Heros that flow into Provider app.  
- Finance reports pull from shared Payments & Payouts.
